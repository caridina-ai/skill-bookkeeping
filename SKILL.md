# 現金記帳

## 工作流程

以下以 `<BOOK>` 代表：

```text
python "<本 SKILL.md 所在目錄>/scripts/book.py"
```

每次依序做四件事：

1. 把使用者的多行要求拆成動作；完整科目清單替換視為一個動作。
2. 每個動作依下方「路由順序」只選第一個命中的類型。
3. 抽出參數並執行對應的 `<BOOK>` 指令；金額、餘額與合計都交給程式計算。
4. 有執行指令時只回傳 stdout，不加解釋、標題或 Markdown。多個動作依原順序回傳每次 stdout，不能只回最後一筆；需要澄清或遇到不支援的歷史倒推時，才使用規定的文字回覆。

補充規則：

- 所有文字參數都加雙引號；日期時間必須是單一參數，例如 `--date "2026-07-09 14:55"`。
- 參數足夠且分類明確時直接執行。只有缺少必要資訊或分類真的有歧義時才詢問。
- 不向使用者暴露內部指令或推測副作用；明確要求的操作交給程式驗證，再原樣回傳結果。
- 支出前若不知道目前科目，先執行 `<BOOK> categories`。這個內部查詢不回傳給使用者。
- 搜尋得到多筆後若接著用 rowid 完成修改／刪除，只回傳第二次操作的 stdout；若仍無法選定，才回傳候選清單請使用者確認。

## 路由順序

| 優先 | 使用者語意 | 動作 |
|---:|---|---|
| 1 | 明確提到科目的列出、新增、刪除、清空、替換或更名 | 管理科目 |
| 2 | 刪掉、取消、被失主拿回；記錯、其實是、改成新金額 | `delete` 或 `edit` |
| 3 | 起始／期初餘額，或宣告手上實際現金 | `adjust` |
| 4 | 封存、備份並清空某年以前的舊帳 | `archive` |
| 5 | 查餘額、明細、流水、分帳、分類統計或展開科目 | 查詢與報表 |
| 6 | 提款、入金、薪水、酬勞、退款、撿到或收到錢；不含提款／領款手續費 | `in` |
| 7 | 買東西、吃飯、看診、繳費、旅行或其他花費 | `out` |

多行要求逐行路由並依原順序執行。說明用的開頭不是帳目；完整科目替換清單則整份一次執行。

## 管理科目

| 使用者語意 | 指令 |
|---|---|
| 列出所有科目 | `<BOOK> categories` |
| 新增科目 A | `<BOOK> category-add --name "A"` |
| 刪除科目 A | `<BOOK> category-delete --name "A"` |
| 刪除／清空全部科目 | `<BOOK> category-clear` |
| 把科目 A 更名為 B | `<BOOK> category-rename --from "A" --to "B"` |
| 捨棄原清單，改用完整新清單 | 一次執行 `category-replace`，每科各傳一個 `--name` |

完整替換不可拆成「先清空、再逐筆新增」。刪除、清空或替換若會移除仍被帳目使用的科目，程式會拒絕且不會刪除帳目；直接回傳 stdout。

## 修改與刪除帳目

執行 `edit`／`delete` 這個動作時，用它本身的 `--find` 搜尋，不從報表內容猜 rowid。

- 可重複傳入多個 `--find`；它們是 OR。使用者明說的詞全部原樣傳入，AI 也可補 2–3 個精準同義詞。
- 使用者若提供日期、時間或原金額，也傳入以縮小範圍。
- 0 筆或剛好 1 筆：原樣回傳程式結果。
- 多筆：程式不會改資料，會列出 `rowid、日期時間、科目、品項、金額`。能從使用者原意確定其中一筆時，再用該 rowid 執行第二次；否則回傳候選清單請使用者選擇。
- 多筆後不得再用關鍵字猜「最近一筆」。第二次只能用 `edit --rowid ...` 或 `delete --rowid ...`。
- `edit`／`delete` 不接受 `--month`；使用者只給月份時，月份只能協助從候選清單判斷 rowid。

## 餘額、入金與支出

- `起始餘額 2000`、`期初現金 2000`、`目前現金餘額 1883` 都是絕對餘額校正，直接使用 `adjust --balance`。目標數字不是入金金額。
- `現在餘額多少？`、`還剩多少現金？` 才使用 `balance`。
- 本帳本追蹤手上現金，所以提款、提領、ATM 領款是 `in`；只有提款／領款手續費是支出。note 保留完整說明，例如銀行名稱。
- 支出若只給「現有科目名稱＋金額」，仍直接記帳，不追問品項；note 使用科目名稱去掉尾端的「費」或「金」（例如「買菜金」記為「買菜」）。
- 支出有花費金額時使用 `out --amount`。若只說「買完剩 X」且沒有日期時間，使用 `out --balance` 讓程式倒推。
- 倒推若同時指定日期或時間，不執行，固定回覆：

```text
倒推不支援指定日期或時間。請告訴我這次實際花了多少錢，或移除日期時間後再用目前餘額倒推。
```

## 日期、時間與報表

- `M/D`、月份或相對日期未寫年份時，以目前日期換成完整年份。
- 只有日期：`--date "YYYY-MM-DD"`；有時間：`--date "YYYY-MM-DD HH:MM"`；完全沒寫日期時間就省略 `--date`。
- note 只保留品項／收入說明，移除日期、時間與金額。

| 使用者說法 | 指令 | 輸出方式 |
|---|---|---|
| 分帳明細、分類明細、科目明細、依科目明細 | `category-detail` | 依科目分組列出每筆 |
| 展開一個或多個科目 | `expand` | 只列指定科目 |
| 明細、流水、流水帳、依日期、某天花了什麼 | `detail` | 依日期時間排列 |
| 分帳、分類帳、分類統計、科目統計、支出統計、月報、結算 | `report` | 依科目加總 |

四種報表都可重複 `--month` 查多個月份，或重複 `--date` 查多個日期，但同一查詢不得混用月份和日期。`expand` 另可重複 `--category`；科目與期間順序照使用者說法。

報表沒指定期間時省略期間旗標，程式使用當月。封存整個 2026 年時，截止日是下一年第一天：`archive --before "2027-01-01"`。

## 指令模板

只能使用以下形狀：

```text
<BOOK> categories
<BOOK> category-add --name "<科目>"
<BOOK> category-delete --name "<科目>"
<BOOK> category-clear
<BOOK> category-replace --name "<科目1>" --name "<科目2>" [...]
<BOOK> category-rename --from "<舊名>" --to "<新名>"

<BOOK> balance
<BOOK> adjust --balance <目標餘額>
<BOOK> in --note "<說明>" --amount <正整數> [--date "<YYYY-MM-DD 或 YYYY-MM-DD HH:MM>"]
<BOOK> out --category "<現有科目>" --note "<品項>" --amount <正整數> [--date "<YYYY-MM-DD 或 YYYY-MM-DD HH:MM>"]
<BOOK> out --category "<現有科目>" --note "<品項>" --balance <剩餘餘額>

<BOOK> edit --find "<關鍵字1>" [--find "<關鍵字2>" ...] [--amount <原金額>] --to <新金額> [--date "<YYYY-MM-DD 或 YYYY-MM-DD HH:MM>"]
<BOOK> edit --rowid <候選 rowid> --to <新金額>
<BOOK> delete --find "<關鍵字1>" [--find "<關鍵字2>" ...] [--amount <原金額>] [--date "<YYYY-MM-DD 或 YYYY-MM-DD HH:MM>"]
<BOOK> delete --rowid <候選 rowid>

<BOOK> report [--month "YYYY-MM" ... | --date "YYYY-MM-DD" ...]
<BOOK> category-detail [--month "YYYY-MM" ... | --date "YYYY-MM-DD" ...]
<BOOK> expand --category "<科目>" [--category "<科目>" ...] [--month "YYYY-MM" ... | --date "YYYY-MM-DD" ...]
<BOOK> detail [--month "YYYY-MM" ... | --date "YYYY-MM-DD" ...]
<BOOK> archive --before "YYYY-MM-DD"
```

## 全新帳本的預設科目

全新資料庫依下表順序建立 16 科，開箱即可使用。這些分類提示只在該科目目前仍存在時適用；使用者若已刪除、更名或替換科目，必須改用目前清單，不可偷偷恢復預設科目或自行新增科目。

| 科目 | 常見品項 |
|---|---|
| 外食費 | 早餐、午餐、晚餐、便當、飲料、冰茶、咖啡、餐廳、外送 |
| 買菜金 | 買菜、蔬菜、水果、全聯、超市、菜市場、食材、雞蛋、肉 |
| 民俗節日 | 紅包、拜拜、金紙、粽子、月餅、年貨 |
| 居住費 | 房租、水費、電費、瓦斯費、網路、電話、修繕 |
| 管理費 | 社區管理費、大樓管理費 |
| 交通費 | 捷運、公車、火車、高鐵、計程車、加油、停車、悠遊卡加值 |
| 車稅險 | 牌照稅、燃料稅、汽機車保險、驗車、公路使用養護管理費 |
| 保健費 | 藥、看診、掛號、牙科、醫院、診所、維他命、健檢 |
| 治裝費 | 衣服、褲子、鞋子、包包、飾品 |
| 精進金 | 書、課程、學費、訂閱、軟體、線上課 |
| 旅遊金 | 旅遊、住宿、飯店、門票、機票、伴手禮 |

| 奉獻 | 捐款、香油錢、奉獻、樂捐 |

若品項無法可靠歸入目前任何科目，先詢問使用者；不要硬塞進雜費。
